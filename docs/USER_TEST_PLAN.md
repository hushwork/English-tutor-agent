# Camera Tutor — 用户测试计划 (Alpha Family Testing)

> **目标**: 在 3-5 个有 3-8 岁孩子的家庭中进行 1-2 周的实际使用测试
> **目标**: 验证产品假设、发现体验问题、收集家长反馈、为下一步产品化提供数据

---

## 1. 测试目标

### 1.1 核心假设验证

| # | 假设 | 验证方式 | 成功标准 |
|---|------|---------|---------|
| H1 | 孩子愿意主动跟 Emma 说话 | 统计开口次数 + 家长观察 | 每天 ≥ 2 次主动开口 |
| H2 | 在真实家庭噪声环境下，语音识别可用 | SNR 测量 + 理解准确率 | 理解准确率 > 70% |
| H3 | Emma 不打断孩子的专注时刻 | 家长观察 + 日志分析 | 家长反馈"没有打扰" |
| H4 | 家长觉得有价值，愿意付费 | 付费意愿调查 | ≥ 60% 愿意付费 |
| H5 | 孩子英语能力在 1-2 周内观察到变化 | 前/后对比 + 家长反馈 | ≥ 50% 家长观察到进步 |

### 1.2 次要验证目标

- LED 光环语言是否能被孩子理解
- 物理镜头盖是否被家长视为信任信号
- 每日报告的家长满意度
- 设备在 7×24 运行中的稳定性
- 硬件散热和噪音在实际家庭中的表现

## 2. 招募标准

### 2.1 入选条件

| 条件 | 要求 |
|------|------|
| **孩子年龄** | 3-8 岁 |
| **英语基础** | 不限（零基础到有基础均可，需要覆盖不同水平） |
| **家庭语言环境** | 以中文为主（非英语家庭） |
| **家长英语水平** | 不限（最好有"自己无法辅导"的真实痛点） |
| **家中空间** | 有固定的书桌/活动区，可安装摄像头 |
| **WiFi** | 家庭 WiFi 稳定 |
| **设备使用** | 可接受房间内有摄像头设备（有物理镜头盖）|
| **参与意愿** | 愿意使用 1-2 周，并每天填写简短反馈 |

### 2.2 推荐的 5 个典型用户画像

| # | 孩子 | 年级 | 英语水平 | 家长期望 | 特殊关注 |
|---|------|------|---------|---------|---------|
| 1 | 小美，4 岁 | 幼儿园中班 | 零基础，只认识 ABC | 培养兴趣和语感 | 年龄最小的测试者，关注注意力和接受度 |
| 2 | 小明，6 岁 | 小学一年级 | 认识一些单词，不会说 | 能开口说简单英语 | 典型"会读不会说"的中国学生 |
| 3 | 小华，8 岁 | 小学三年级 | 有课外班基础，B1- | 替代/补充每周 1 次的线下班 | 已有基础，关注学习效果 |
| 4 | 小红，5 岁 | 幼儿园大班 | 上了半年英语班 | 增加日常英语接触 | 双职工家庭，祖辈带娃 |
| 5 | 小刚，7 岁 | 小学二年级 | 零基础，对英语抵触 | 希望能让英语变得有趣 | ADHD 倾向，关注注意力 |

## 3. 测试流程

### 3.1 时间线

```
Day -3: 招募确认 → 签署知情同意书 → 设备寄送/安装
Day -1: 设备安装指导（视频通话 30 分钟）
Day 0:  设备就位，开始 24 小时运行测试（硬件稳定性）
Day 1:  正式使用开始
Day 3:  中期电话回访（15 分钟）
Day 7:  第一周问卷收集
Day 14: 测试结束 → 设备回收 → 最终深度访谈（30 分钟）
Day 21: 数据分析 + 报告撰写
```

### 3.2 安装指导清单

```
[ ] 设备固定在孩子书桌/活动区前上方 45°，高度 40-60cm
[ ] 摄像头覆盖孩子主要活动区域（书桌 + 地面活动区）
[ ] 音箱距离孩子活动区 1-2 米
[ ] 物理镜头盖可以顺畅滑动
[ ] LED 光环可见但不刺眼
[ ] WiFi 连接正常，家长 Dashboard 可访问
[ ] 家长了解镜头盖的使用方式（隐私开关）
[ ] 设置孩子的年龄和英语水平
[ ] 设置禁用时段（如果孩子有固定做作业时间）
```

### 3.3 家长每日任务

| 时间 | 任务 | 耗时 |
|------|------|------|
| 每晚 | 查看 Dashboard 每日报告 | 2 分钟 |
| 每晚 | 填写简短反馈（3 个问题）| 2 分钟 |
| 随时 | 记录"孩子说英语的惊喜时刻"（可选）| 按需 |

### 3.4 每日反馈问题

```
Q1: 今天孩子跟 Emma 说话了吗？
    [ ] 没说话  [ ] 说了 1-2 次  [ ] 说了 3-5 次  [ ] 说了很多次

Q2: 今天 Emma 有没有在不该说话的时候说话？（如：做作业时、专注时）
    [ ] 没有打扰  [ ] 偶尔打扰  [ ] 经常打扰

Q3: 今天你对孩子的英语学习有什么观察？
    ________________________________________
```

## 4. 数据收集

### 4.1 定量数据（自动采集）

| 指标 | 数据来源 | 频率 |
|------|---------|------|
| 每日英语输入句数 | 系统日志 | 每日 |
| 孩子开口次数 | VAD + 语音检测 | 每日 |
| 介入频率（Emma 主动说话的次数）| 决策引擎日志 | 每日 |
| 专注时段（不打扰的时长）| 场景分析器 | 每日 |
| 互动持续时间 | 对话系统 | 每次 |
| 识别准确率（Emma 理解孩子语音的比例）| 手动标注抽样 | 每周抽取 20 条 |
| 绘本识别准确率 | 人工验证 | 测试期间 |
| 设备在线时长 | 系统日志 | 每日 |
| 系统资源（CPU/GPU/温度/功耗）| Jetson 监控 | 每小时 |

### 4.2 定性数据（家长反馈）

| 方法 | 内容 | 时机 |
|------|------|------|
| 每日短问卷 | 3 个选择题 + 1 个开放式 | 每天 |
| 中期电话回访 | 15 分钟半结构化访谈 | Day 3-4 |
| 终期深度访谈 | 30 分钟（录屏/录音） | Day 14 |
| 孩子观察笔记 | 家长记录"孩子跟 Emma 的互动瞬间" | 随时 |

## 5. 终期访谈提纲

### 5.1 孩子端

```
1. 孩子喜欢 Emma 吗？怎么知道的？
2. 孩子有没有主动跟 Emma 说话的时刻？是什么情况？
3. 孩子有没有模仿 Emma 说英语？是哪句话？
4. 孩子有没有因为 Emma 说英语而对某些东西更感兴趣了？
5. 孩子有没有说过"不要"/害怕/讨厌 Emma？为什么？
6. 孩子注意到了 LED 光环吗？知道不同颜色什么意思吗？
7. 家长观察到孩子的英语有什么变化吗？
```

### 5.2 家长端

```
8. 使用时最大的惊喜是什么？最大的失望是什么？
9. 你在什么场景下觉得这个产品最有用？
10. 你在什么场景下觉得它没用/甚至烦人？
11. Dashboard 对你有用吗？你最看哪个数据？
12. 镜头盖让你觉得安心吗？你用过几次？
13. 如果有朋友问你这是什么，你会怎么说？
14. 如果定价 ¥1,299（设备）+ ¥29/月，你会买吗？
    [如果不会] 多少钱你会买？
15. 你希望这个产品还能做什么？
```

### 5.3 技术端

```
16. 设备是否稳定？有没有出过故障？
17. 音箱声音是否清晰？音量够吗？
18. 有没有误触发（不该说话时说了）？
19. 有没有漏触发（该说话时没说）？
20. 夜间是否有噪音/光污染？
```

## 6. 成功标准

### 6.1 必须通过（否则产品需要重大调整）

| 标准 | 指标 | 门槛 |
|------|------|------|
| 孩子接受度 | 孩子不排斥 Emma | 5/5 家庭 |
| 不打扰专注 | Emma 没有在专注时说话 | 4/5 家庭 |
| 语音可用 | 孩子语音能被理解 | > 60% |
| 家长安全感 | 镜头盖被认可 | 4/5 家庭 |
| 硬件稳定 | 7×24 运行无重大故障 | 5/5 设备 |

### 6.2 想要通过（表明产品方向正确）

| 标准 | 指标 | 门槛 |
|------|------|------|
| 孩子主动开口 | 每天 ≥ 2 次 | 3/5 家庭 |
| 付费意愿 | 愿意付费 | 3/5 家庭 |
| 英语进步 | 1 周内观察到变化 | 3/5 家庭 |
| 推荐意愿 | 愿意推荐给朋友 | 4/5 家庭 |
