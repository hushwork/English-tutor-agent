## 1. 理论基础回顾

Camera Tutor 是 english-tutor 项目在幼儿场景的自然演化。english-tutor 的核心理论框架详见 `PRODUCT_DOC.md`，在此总结关键概念：

### 1.1 LLM 三阶段类比（成人）

```
Phase 1: Pretraining           →    Immersion Mode（浸泡模式）
  类比: 婴儿在母语的环境中，不识字，不学语法，只是听
  方法: 大量可理解的多感官输入，零输出压力

Phase 2: Instruction Tuning    →    AI 对话（Chat Mode）
  类比: 学龄儿童在学校系统学习
  方法: 有目的的口语练习、纠错、话题讨论

Phase 3: LoRA Fine-tuning      →    SM-2 间隔复习（Review Mode）
  类比: 成人在 L1 基础上嫁接 L2
  方法: 词汇记忆、间隔重复、有意识的语法学习
```

### 1.2 程序式记忆 vs 声明式记忆

| | 声明式记忆（成人依赖） | 程序式记忆（婴儿依赖） |
|---|---|---|
| **脑区** | 海马体 | 基底神经节、小脑 |
| **特点** | 脆弱，需要意识调用 | 鲁棒，自动激活 |
| **类比** | 忘了中午吃什么 | 不会忘记怎么骑车 |
| **睡眠后** | 不巩固则遗忘（艾宾浩斯曲线） | 睡眠中自动强化 |

### 1.3 关键洞察

成人学英语最大的问题：**用 Phase 3 的方法（背单词、学语法 = LoRA 微调）去实现 Phase 1 应该完成的目标（直觉性理解 = 预训练）**。

## 2. 为什么幼儿学习英语更接近"母语习得"

### 2.1 三个关键差异

| 维度 | 幼儿 (3-8 岁) | 成人 |
|------|-------------|------|
| **海马体** | 未完全发育成熟 | 完全成熟 |
| **默认记忆系统** | 程序式记忆为主 | 声明式记忆为主 |
| **L1 干扰** | 中文网络仍在建设中，尚可塑 | 中文网络已固化数十年 |
| **语音辨别** | 能辨别所有语言的音素差异 | 只能辨别已习得语言的音素 |
| **学习机制** | 隐性、无意识、统计学习 | 显性、有意识、规则学习 |
| **情感过滤** | 较低（不怕犯错、不怕被笑） | 高（焦虑、面子、自我意识） |

### 2.2 语言的"关键期假说"再审视

经典的关键期假说认为语言习得能力在青春期后急剧下降。但最新的神经科学表明：

- **不是"关闭"而是"竞争加剧"**：L1（中文）网络越强大，对 L2 输入的抑制越强
- **幼儿海马体不成熟这个"缺陷"反而是优势**：因为海马体不能用，他们被迫使用基底神经节的程序式学习
- **语音敏感期是真实的**：3-7 岁是语音辨别的关键窗口，之后辨别非母语音素的能力持续下降

### 2.3 对中国幼儿的启示

中国幼儿学英语的起点不是"零"——他们已经或正在建立强大的中文网络。但不同于成人：

- 中文网络**仍在建设中**（3-6 岁儿童的中文词汇量还在快速增长）
- 大脑的**可塑性仍然很高**
- 他们**默认使用隐性学习机制**（不像成人已经切换到显性学习）

这意味着：**如果提供足够量的可理解英语输入，幼儿可以建立"接近母语"的英语语音和语感基础，而不需要像成人那样依赖翻译和语法规则。**

## 3. 环境浸泡理论：为什么真实世界 > 屏幕

### 3.1 Krashen 的"输入假说"在真实世界中的实现

Stephen Krashen 提出语言习得的关键是 **"可理解输入"（Comprehensible Input）**——学习者理解大意但不一定理解每个词。但 Krashen 没有解决一个问题：**如何持续产生可理解输入？**

- **外教/家长**：可以，但太贵/太累/不可持续
- **动画片/App**：内容固定，不可个性化，脱离孩子当前环境
- **绘本**：可以，但需要有人读，且不会根据孩子的反应动态调整

**Camera Tutor 的方案**：利用孩子所处的**真实环境本身就是"可理解输入"的来源**。孩子正在玩积木 → "Blocks! Red block, blue block!" 孩子天然理解上下文（他在玩的东西），英语只是对已有理解的**语言标注**。

### 3.2 为什么环境输入优于屏幕输入

| | 屏幕（iPad/动画片） | 真实环境（Camera Tutor） |
|---|---|---|
| **情境相关性** | 虚拟情境，可能与孩子此刻无关 | 100% 相关（孩子正在做的事） |
| **注意力质量** | 被动接收，容易"看进去但没听进去" | 主动参与，语言与动作关联 |
| **多感官绑定** | 视觉（屏幕）+ 听觉 (有限) | 视觉 + 听觉 + 触觉 + 本体感觉 |
| **情感联结** | 对角色产生情感，但对语言无感 | 语言与自我体验绑定（"这是我的积木"） |
| **长期记忆** | 弱（孤立在海马体） | 强（多感官绑定 + 情绪标记） |
| **健康** | 久坐、视力疲劳、蓝光 | 正常活动、身体健康 |

### 3.3 多感官绑定理论（Multisensory Binding）

认知神经科学的一个核心发现：**记忆不是存储在单一脑区，而是分布在感知发生时的同一神经网络中。**

当孩子拿起一个红色的积木，同时听到 "Red block!"，此时激活的网络包括：

```
视觉皮层: 红色、方块形
    +
本体感觉: 手的触觉、积木的重量
    +
运动皮层: 拿起这个动作
    +
听觉皮层: "Red block!" 的声音
    +
前额叶: 注意力集中在这个物品上
    =
一个"红色积木"的分布式多感官表征
```

未来当孩子听到 "red"，这个**整个网络**会被部分激活——不只是"red = 红色"的翻译记忆，而是包含了触觉、动作、情境的完整体验。

这就是**为什么在真实环境中学到的语言更不容易被遗忘**——它不依赖单一的海马体记忆痕迹，而是嵌入在整个感知-运动网络中。

### 3.4 屏幕输入的多感官贫困

动画片里的积木：孩子只能**看**。他不能摸、不能搭、不能推倒。激活的脑区只有视觉皮层 + 部分听觉皮层。网络连接稀疏，容易遗忘。

## 4. 摄像头如何实现"可理解输入 + 1"

### 4.1 i+1 的环境版本

Krashen 的 i+1 原则：输入应该比学习者当前水平稍难一点（"i" 是完全理解，"1" 是新信息）。

Camera Tutor 如何自动产生 i+1：

| 孩子水平 (i) | 摄像头看到的 | 生成的 i+1 输入 |
|-------------|-------------|-----------------|
| 能听懂颜色词汇 | 红色积木 | "A red block! Can you find a blue block?" |
| 能听懂简单指令 | 在收拾玩具 | "Put the car in the box. Good! Now put the doll." |
| 能说单词 | 指着狗说 "dog" | "Yes! A brown dog. The dog is barking. Woof woof!" |
| 能说简单句 | "I draw cat." | "You're drawing a cat! The cat has a long tail. What else?" |

### 4.2 为什么摄像头能做到，而 App 做不到

1. **视觉上下文自动提供 i 的水平估计**：看到孩子在玩什么 → 推断孩子目前能理解什么
2. **物品识别提供"免费词汇教学"**：孩子所有注意力都在那个物品上，此时给英语标签是最自然的
3. **活动识别提供"免费语法教学"**：孩子在画画 → "You are drawing"（现在进行时自然输入，不需要解释语法规则）
4. **情绪识别提供情感合适的输入**：孩子沮丧时不给挑战性输入，孩子开心时可以推高难度

### 4.3 三种语言输入类型

Camera Tutor 提供三种输入混合：

| 类型 | 例子 | 目的 | 占比 |
|------|------|------|------|
| **背景旁白** | "She's building a tower. One block, two blocks..." | 营造英语氛围，不要求回应，类似"白噪音" | 70% |
| **互动提问** | "What color is this?" / "Can you find something red?" | 检验理解，鼓励输出 | 20% |
| **直接教学** | "Say it with me: red, blue, green. Red!" | 明确的语言教学（仅在孩子表现出学习意愿时） | 10% |

## 5. 三种角色背后的认知科学

### 5.1 📹 学习记录仪 — 安全基地（Secure Base）

依据：**依恋理论（Attachment Theory, John Bowlby）**

儿童需要一个"安全基地"——一个可靠、可预测、不会消失的存在——才能安心探索世界。

Camera Tutor 的默认状态（蓝色呼吸灯，安静观察）提供的就是这种**安全基地感**：孩子知道它在，但它不干扰。这种"非侵入性的存在"是孩子敢于用英语尝试的前提。

相比之下，一个人走过来突然说 "Hello! How are you!"（培训班外教的典型开场），对幼儿来说可能是**警觉信号**而非**学习信号**。

### 5.2 🧑‍🏫 英语导师 — 最近发展区（Zone of Proximal Development）

依据：**Vygotsky 的最近发展区理论**

孩子能独立完成的事情 vs 在有帮助的情况下能完成的事情之间存在一个"最近发展区"（ZPD）。好的教学应该在这个区域内。

Camera Tutor 通过摄像头识别孩子的当前活动，自动定位 ZPD：

```
孩子能独立做的           ZPD (最近发展区)         孩子暂时做不了
(不教)                   (Camera Tutor 教这里)    (不教)
    │                                                    │
听懂 "car"          →  能说 "red car"              →  能用英语描述汽车构造
说出颜色            →  能说出完整短语                →  能区分一般现在时和现在进行时
听懂简单指令         →  能执行两步指令               →  能理解复杂条件句
```

### 5.3 🎮 游戏搭子 — 玩中学（Play-Based Learning）

依据：**Montessori 和 Piaget 的游戏即学习理论**

幼儿的认知发展不是在"课堂"中完成的，而是在**游戏**中完成的。假装游戏（Pretend Play）尤其重要——它激活了符号思维，这正是语言的基础。

Camera Tutor 的游戏模式设计为**假装游戏的催化剂**，而非结构化教学游戏：

- "Let's pretend we're animals!" → 孩子用身体动作模仿动物，同时输入英语动物名称
- "I spy something blue..." → 利用已有的"我来猜"游戏框架，只需要替换语言
- "Simon says touch your nose!" → TPR（Total Physical Response），语言+动作绑定

这些游戏都不需要屏幕，不需要规则说明，依赖的是孩子已有的游戏认知框架。

## 6. 为什么"不打扰"比"教"更重要

### 6.1 注意力即学习

神经科学的研究反复确认：**深度专注（Flow State）时的学习效率是分散注意时的 5-10 倍。**

对幼儿来说，深度专注尤其宝贵——因为他们的大脑正在建立基础神经连接，被打断的代价不只是"少学了这一会儿"，而是**破坏了正在形成的神经回路**。

Camera Tutor 的设计哲学是：

> 宁可错过 10 次教学机会，也不打断 1 次深度专注。

### 6.2 介入条件的认知基础

| 状态 | 检测方式 | 行动 |
|------|---------|------|
| **深度专注** | 头部稳定、视线固定、手部有目的性活动、超过 3 分钟 | **绝对沉默**（LED 蓝色呼吸） |
| **浅层活动** | 活动频繁切换、漫无目的、< 1 分钟/活动 | **可介入**（等到活动切换间隙） |
| **社交邀请** | 看向摄像头、举起物品、主动说话 | **立即回应** |
| **挫败/无聊** | 唉声叹气、扔东西、趴桌子、长时间不动 | **切换为游戏模式或休息引导** |
| **多人在场** | 检测到 2+ 人声音 | **降低介入频率**（不打扰亲子互动/同伴互动） |

### 6.3 频率限制的神经科学依据

海马体的长时程增强（LTP — Long-Term Potentiation）需要时间巩固。如果信息输入过于频繁，会发生"干扰"（Interference），新信息覆盖旧信息。

对于幼儿，**每次教学介入后至少间隔 10-15 分钟**，让刚输入的英语在安静环境中自然回响和巩固。这就是为什么我们设计**每小时最多 3-5 次主动介入**。

### 6.4 沉默时段的价值

沉默不只是"没有在学习"——沉默是**巩固**。

在语言输入后的一段安静中，大脑在做什么：
- **回放（Replay）**：海马体在快速回放刚才的听觉体验
- **模式提取**：基底神经节在统计音素/词汇的出现模式
- **巩固（Consolidation）**：从海马体的临时存储转移到皮层的长期存储

这就是为什么睡觉对语言学习至关重要——睡眠中这些过程被**加速和强化**。而 Camera Tutor 不需要取代睡眠，只需要**不占用巩固所需的安静时间**。

## 7. 设计决策的认知科学依据

| 设计决策 | 认知科学依据 |
|---------|------------|
| **无屏幕** | 多感官绑定理论：触觉/本体感觉缺失 → 记忆弱化；AAP/WHO 屏幕时间建议 |
| **环境优先** | 情境学习理论（Lave & Wenger）：学习发生在情境中，不可脱离 |
| **摄像头观察** | 联合注意（Joint Attention）：教-学双方关注同一对象是语言习得的基础 |
| **物理镜头盖** | 自主性（Autonomy）需求：人（包括幼儿）需要感知对环境的控制 |
| **Emma 简化英语** | 儿童指向言语（Child-Directed Speech）：拖长元音、高音调、短句 → 促进语言习得 |
| **重述式纠错** | 消极证据（Negative Evidence）：不说"你错了"，而是给出正确版本 |
| **LED 光环语言** | 边缘注意力（Peripheral Attention）：不占用中央视觉通道，不打断当前任务 |
| **三种角色切换** | 自我决定理论（SDT）：自主性（游戏搭子）、胜任感（导师引导）、关联性（记录仪的安全基地） |
| **英文 only** | 直接法（Direct Method）：不在脑内建立"翻译中间层"，直接建立 sound→meaning 映射 |
| **间隔重复 (SM-2)** | 间隔效应（Spacing Effect）：分散学习比集中学习效率高 2-3 倍 |

### 7.1 从 english-tutor 三段式到 Camera Tutor 三段式

```
english-tutor Immersion 三阶段          Camera Tutor 环境浸泡三阶段

Pass 1: 👂 Sound First               Pass 1: 👀 Observe
  纯音频，不看文字                      静默观察孩子活动，不做任何干预
  目的: 建立声音→节奏→情感的直觉         目的: 理解上下文，定位 ZPD
  
Pass 2: 👁️ Read Along                Pass 2: 🗣️ Describe
  文字+音频同步                         用简单英语描述孩子正在做的事
  目的: 连接文字符号与声音              目的: 提供 i+1 可理解输入

Pass 3: 🧘 Eyes Closed               Pass 3: 🤝 Interact
  纯音频再听一次                        提问/邀请互动/等待回应
  目的: 巩固、内化                      目的: 检验理解、鼓励输出、增强记忆绑定
```

## 8. 视觉语音与发音教学

> Emma 的脸不是装饰品。它是基于 50 年视觉语音感知研究的发音教学工具。

### 8.1 核心研究

| 研究 | 发现 | 对我们的意义 |
|------|------|------------|
| **McGurk & MacDonald** (Nature, 1976) | 听觉和视觉语音在脑干层面融合。音频"ba"+嘴型"ga"=大脑听到"da"。**视觉是语音感知的一部分，不是附加品** | 嘴型不是辅助——是语音输入的另一半 |
| **Kuhl & Meltzoff** (Science, 1982) | 2-4 个月婴儿就能匹配听到的元音和看到的嘴型。这种跨模态匹配是**先天的** | 孩子不需要"学会"看嘴型——他们天生就会 |
| **Hazan et al.** (2006) | L2 学习者从视觉语音中获益 3-5 倍于母语者。纯听辨 72% → 加视觉 89%（+17%） | 对中国孩子的价值远超对英语母语儿童 |
| **Hardison** (2003) | 视觉线索是 L2 学习者未建立音素分类前的**锚点** | Emma 的嘴型是音素学习的"训练轮" |
| **Rizzolatti & Arbib** (Trends in Neurosciences, 1998) | 镜像神经元：**看到嘴型 → 激活自己说话的相同运动皮层** | 看嘴型 = 大脑在"练习"发音 |

### 8.2 为什么对中国孩子特别重要

汉语和英语的**音素库存（phoneme inventory）**有系统性差异。以下英语音素在汉语中**不存在**或**区分方式不同**——视觉通道是学习正确发音的主要途径：

| 音素 | 汉语问题 | 视觉教学价值 | 孩子应该看什么 |
|------|---------|------------|--------------|
| **/θ/ /ð/** (think, this) | 汉语无齿间音。孩子用 /s/ /z/ 替代 → "sink" for "think" | ⭐⭐⭐⭐⭐ | 舌尖伸出上下齿之间（最可见的英语音素） |
| **/f/ /v/** (five, very) | /v/ 不存在。孩子用 /w/ 替代 → "wery" for "very" | ⭐⭐⭐⭐⭐ | 上齿咬下唇 vs 双唇圆拢 → /v/ vs /w/ 一眼可辨 |
| **/w/ /r/** (wet, red) | /w/ 和 /v/ 混淆，/r/ 卷舌方式与汉语不同 | ⭐⭐⭐⭐ | 双唇圆拢前突 → wet vs vet 的视觉区分 |
| **/æ/** (cat, apple) | 汉语无此开口度。孩子用 /e/ 或 /a/ 替代 → "bed" for "bad" | ⭐⭐⭐⭐ | 大开口 + 嘴角拉开 → 比 /e/ 明显更开 |
| **/iː/ /ɪ/** (sheep/ship) | 汉语无长短元音对立 | ⭐⭐⭐ | /iː/ 嘴角更向两边，肌肉更紧张 |
| **/uː/ /ʊ/** (pool/pull) | 汉语无此对立 | ⭐⭐⭐ | 圆唇程度不同：更突/更圆 vs 微圆 |
| **/m/ /n/ /ŋ/** (mom/no/sing) | 尾鼻音 /ŋ/ 常被省略 | ⭐⭐⭐ | /m/ 闭唇 vs /n/ 舌尖顶上颚 vs /ŋ/ 舌后抬起 |

### 8.3 Emma 的面部设计如何服务于教学

```
Emma 不说话时:
  屏幕熄灭（AMOLED 纯黑）→ 不干扰孩子注意力

Emma 说含 /θ/ 的词时:
  面部出现 → 嘴型夸张 → 舌尖明显伸出 → 孩子下意识模仿
  ↓
  镜像神经元激活 → 运动皮层"练习" → 未出声但大脑在学习

Emma 说含 /v/ 的词时:
  嘴型夸张 → 上齿咬下唇清晰可见 → 与 /w/ 的圆唇形成鲜明对比
  ↓
  孩子形成视觉记忆："咬唇的是 /v/，圆唇的是 /w/"
```

### 8.4 实现映射

代码 `avatar.py` 中 `CHINESE_L2_HOTSPOTS` 表定义了 6 个对中国孩子教学价值最高的嘴型，每个 `Viseme` 枚举带有 `teaching_priority`（1-5）和 `needs_tongue_visible` 属性。SVG 渲染器在渲染 `/θ/` `/t/` `/d/` 时会额外绘制舌头的椭圆。

`EmmaAvatar.get_teaching_guidance(viseme)` 方法返回孩子友好的发音指导提示，如：
- `/θ/` → "Put your tongue between your teeth! Like this!"
- `/v/` → "Bite your lip gently! Top teeth on bottom lip!"
