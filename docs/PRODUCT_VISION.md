# Camera Tutor — 产品愿景与功能定义

> **一句话定位**: 一个放在孩子房间前上方的摄像头+投影组合，不依赖屏幕，用英语观察、陪伴、教导你的孩子。平时隐于环境，互动时桌面变教室。

## 1. 产品哲学

### 1.1 核心理念

> **"English just happens."**

我们不认为英语应该被"学"——它应该被"活"。就像孩子学母语时没有意识到自己在"学中文"，他们只是在生活：听妈妈说话、跟爸爸讲今天发生的事、一个人自言自语地玩玩具。

Camera Tutor 的目标是在中国家庭里创造**最小可行英语环境**（MVEE — Minimum Viable English Environment）。不是替代家长，不是替代外教，而是在家长无法用英语交流的时刻、外教不在的时刻——也就是每天的大部分时刻——提供一个"英语背景音 + 偶尔互动的英语朋友"。

### 1.2 设计原则

| 原则 | 说明 |
|------|------|
| **远离交互屏** | 不做触摸屏、不做可滑动/点击的 UI。显示只用于 Emma 面部+嘴型(Mid 版)或桌面投影(Pro 版)，不说话时完全熄灭。 |
| **环境优先** | 不创造虚拟世界，而是赋能真实世界。孩子玩玩具、画画、看书都是学习材料。 |
| **尊重专注** | 孩子深度专注时绝不打扰。我们不是"推送内容"的产品，我们是"等待可教时刻"的观察者。 |
| **边缘智能** | 数据不出家门。摄像头画面只在本地处理。家长可以拔掉电源、盖上镜头盖——完全掌控。 |
| **"爸爸/妈妈看得见"** | 家长不是被排除在外的。每日报告、精彩时刻、学习进度——透明、可信。 |

### 1.3 我们对"屏幕"的态度

我们反对的不是"显示"本身，是**孩子低头操作的交互屏**：

```
iPad/手机 (我们反对的):           Camera Tutor Mid/Pro (我们做的):
  ↓                                    ↓
触控交互 → 可打开游戏/视频          纯显示，不可触摸
被动消费 → 注意力被劫持             只在说话时亮 → 沉默时全黑
蓝光长时间照射                      暗色背景 + 仅面部发光
孩子: "我想玩iPad"                  孩子: 不看它也没关系
                                    (语音才是主要通道)
```

**关键区分**: 摄像头看到孩子在玩积木 → Emma 说英语 + 屏幕上显示 Emma 的嘴型 → 孩子的注意力仍在积木上。屏幕是"视觉辅助"，不是"交互中心"。

### 1.4 与现有 english-tutor 项目的关系

Camera Tutor 是 `english-tutor` 项目的**自然演化**：

```
english-tutor (成人/青少年自学工具)
    │
    ├── Immersion 三阶段 → 适配为"环境浸泡"(Pass1: 观察 → Pass2: 描述 → Pass3: 互动)
    ├── Emma 导师角色 → 演化为"Emma 幼儿版"(更慢、更夸张、更鼓励)
    ├── SM-2 间隔重复 → 完全复用(词汇复习不区分年龄段)
    ├── Memory/Stats → 扩展为"家长报告引擎"
    └── FastAPI Web Server → 家长端 Dashboard 的基石
```

## 2. 目标用户画像

### 2.1 主要用户：孩子（3-10 岁）

| 年龄分段 | 特征 | 英语目标 |
|---------|------|---------|
| **3-5 岁（学龄前）** | 不能阅读、注意力 5-10 分钟、喜欢重复、通过动作学习 | 语音语调培养、基础词汇（颜色、数字、动物）、简单指令反应 |
| **6-8 岁（低年级）** | 开始认字、注意力 10-20 分钟、喜欢游戏/比赛、开始有社交意识 | 简单对话、绘本跟读、Show & Tell 描述物品 |
| **9-10 岁（高年级）** | 有基础词汇量、能阅读短句、有独立兴趣、可能已有课外班 | 场景对话、绘本精读、发音纠正、初级写作辅助 |

### 2.2 购买决策者：家长

| 画像 | 特征 | 核心诉求 |
|------|------|---------|
| **焦虑型妈妈（主力）** | 30-40 岁，一线/新一线城市，孩子 3-8 岁，自己英语一般，重视教育 | "我帮不了孩子学英语，但我可以给他买个老师" |
| **高知家长（早期用户）** | 有海外背景或英语好，认可"环境浸泡"理念，追求差异化教育 | "不是又上培训班，是给孩子创造环境" |
| **忙碌爸/妈** | 双职工，没时间接送培训班，祖辈带娃 | "我不在家时，孩子也能接触英语" |

### 2.3 使用场景（决策者的时间线）

```
早上 7:30  孩子起床，在自己房间玩一会儿
           Emma: "Good morning! Did you sleep well?"
           孩子: (含糊地) "Morning!"
           场景: 简单的英文早安对话，2-3 个来回

下午 4:30  孩子放学回家，在书桌上画画
           Emma: (看到画纸上的内容) "You're drawing a rainbow! 
                 Red, orange, yellow, green, blue, purple!"
           场景: 旁白式英语描述，不要求回应

晚上 7:00  孩子拿出一本英文绘本翻看
           Emma: "Oh, The Very Hungry Caterpillar!
                 Monday — he ate one apple. Tuesday — two pears..."
           场景: 绘本共读，翻页即讲

晚上 8:30  家长在手机上查看今日报告
           "今天接触了 47 个英语句子，主动开口 12 次，
            学了 3 个新词：rainbow, caterpillar, apple"
           场景: 家长 Dashboard 每日摘要
```

## 3. 三种角色定义

### 3.1 角色切换逻辑

```
                    ┌──────────────────────────────┐
                    │         📹 学习记录仪          │
                    │      (默认状态，后台持续)        │
                    │  记录专注/活动/语言接触          │
                    └──────────┬───────────────────┘
                               │
                 ┌─────────────┼─────────────┐
                 ▼             ▼             ▼
        孩子主动开口      可教时刻检测      孩子呼叫游戏
                 │             │             │
                 ▼             ▼             ▼
        🧑‍🏫 英语导师      🧑‍🏫 英语导师      🎮 游戏搭子
        "你说话了！"      "看到绘本了！"    "来玩吧！"
```

### 3.2 📹 学习记录仪（默认状态，90% 的时间）

| 属性 | 详情 |
|------|------|
| **状态** | 安静、旁观、记录 |
| **做什么** | 识别孩子当前活动（看书/画画/玩玩具/做作业）、追踪专注时长、识别情绪状态、记录英语语言接触量 |
| **不做什么** | 不说话、不打扰、不录像存储 |
| **产出** | 每 30 分钟一条时间线日志 → 汇总为每日报告 |
| **LED 光环** | 微弱的蓝色呼吸灯（"我在，但不打扰"） |

### 3.3 🧑‍🏫 英语导师（可教时刻激活，~8% 的时间）

| 属性 | 详情 |
|------|------|
| **触发条件** | 孩子看向摄像头/举起物品/打开绘本/说出英语/主动问"What's this?"/摄像头检测到新物品/活动转换间隙 |
| **做什么** | 用简单英语描述物品/活动、问一个开放性问题（"What color is it?"）、对孩子说的英语给予鼓励和纠正 |
| **干预频率** | 每小时最多 3-5 次主动介入，每次 30 秒到 2 分钟 |
| **退避条件** | 孩子不回应（等 5 秒放弃）、孩子说"不要"/"stop"、检测到专注状态（做作业时完全不打扰）|
| **语言原则** | 只说英语（不翻译成中文）、短句（5-8 个词）、重复 2-3 次、夸张的语调、错误用"重述"纠正而非直接指出 |
| **LED 光环** | 柔和的绿色（"我在说英语，你来吗？"） |

#### 导师交互模式

```
模式 1: Show & Tell（展示与讲述）
    孩子举起一个玩具车 →
    Emma: "A car! A red car! Vroom vroom! 
           Do you want to show me more toys?"

模式 2: Picture Book（绘本共读）
    孩子翻开一页 The Very Hungry Caterpillar →
    Emma: "Look! A caterpillar! 
           He's very hungry. What did he eat?"

模式 3: Activity Narration（活动旁白）
    孩子在搭积木 →
    Emma: "One block, two blocks, three blocks! 
           What color is this one?"

模式 4: Question Time（问答时间）
    孩子主动问"What's this?" →
    Emma: "That's a dinosaur! A T-Rex! 
           Can you say 'dinosaur'?"

模式 5: Correction Loop（纠错循环）
    孩子: "I have two foot."
    Emma: "Wow, you have two feet! 
           One foot, two feet. Good job!"
    (不直接说"你错了"，而是给出正确版本)
```

### 3.4 🎮 游戏搭子（受邀或检测到无聊时，~2% 的时间）

| 属性 | 详情 |
|------|------|
| **触发条件** | 孩子说"Let's play!"或"陪我玩"、检测到漫无目的/无聊状态（5 分钟以上无明确活动）、家长设置的游戏时间 |
| **游戏类型** | I Spy ("I spy something red...")、Simon Says ("Simon says touch your nose!")、Counting Game ("Let's count all the blue things!")、Rhyme Time ("What rhymes with cat?")、Movement Break ("Jump three times!")|
| **持续时间** | 3-8 分钟，自然衰减（不强制结束） |
| **LED 光环** | 跳动的暖黄色/橙色（"来玩！"） |

## 4. 核心场景

### 场景 1：自由玩耍旁白（最高频）

```
👧 3 岁，在地垫上玩积木

Emma (检测到积木): "Blocks! You have blocks!"
👧 继续玩，没理
Emma (等 30 秒，再次尝试): "Can you build a tower?"
👧 开始搭高
Emma: "One, two, three, four, five! A tall tower!"
👧 (开心) "Tall!"
Emma: "Yes! Very tall! Can you make it taller?"

● 记录: 接触 8 句英语，开口 1 次，词汇: blocks/tower/tall
```

### 场景 2：绘本共读

```
👧 6 岁，拿出一本英文绘本，坐到摄像头前的地垫上

Emma (识别封面): "Brown Bear, Brown Bear! I love this book!"
👧 翻开第一页
Emma (识别页面): "Brown Bear, Brown Bear, what do you see? 
                  I see a red bird looking at me."
👧 跟着: "Red bird..."
Emma: "Yes! Red bird! Can you find something red in your room?"
👧 环顾四周，指着红色铅笔
Emma: "That's right! A red pencil! 
       Let's read the next page?"

● 记录: 绘本共读 8 分钟，开口 5 次，跟读完整度 60%
```

### 场景 3：作业陪伴（只旁观，不打扰）

```
👧 8 岁，在书桌前做数学作业

Emma (检测: 书写姿势 + 作业本/习题册): 静音
    (LED 蓝色呼吸灯，表示"我在，但不打扰")
    
    15 分钟后...
    
👧 开始玩橡皮、东张西望
Emma (检测: 注意力分散 5 分钟): 
    "Hey, do you want to take a break? 
     Let's stretch our arms up high!"
👧 伸懒腰
Emma: "Good stretch! Now you can finish your homework. 
       Almost done?"

● 记录: 专注 15 分钟 → 走神 5 分钟 → 休息引导 → 情绪中性
```

### 场景 4：Show & Tell（最互动）

```
👧 5 岁，刚画完一幅画，举起来给摄像头看

Emma: "Wow! You drew something! 
       Is that a sun? A big yellow sun!"
👧 "Yes! And this is a cat!"
Emma: "A cat! Is it your cat? What's its name?"
👧 "Mimi!"
Emma: "Mimi the cat! Does Mimi like to play?"
👧 点头
Emma: "I bet you and Mimi are good friends. 
       Can you draw a fish for Mimi?"

● 记录: Show&Tell 5 分钟，开口 8 次，自发性开口 3 次
```

### 场景 5：睡前故事（家长设置的例行程序）

```
家长在 App 设置：每晚 8:30 进入"睡前模式"

8:30
Emma (声音变得柔和): "It's almost bedtime. 
                       Let's listen to a story.
                       Close your eyes and listen..."

Emma 开始讲一个 2 分钟的短故事，语速缓慢，句子很短
    声音越来越轻，最后说 "Good night. Sweet dreams."
    LED 变为微弱的紫色 → 3 分钟后熄灭

● 记录: 睡前模式完成，故事听完
```

### 场景 6：投影 AR 单词匹配游戏（Pro 版）

```
投影在桌面投射 3 张图: apple, car, dog
摄像头看到孩子的手悬停在桌面上方

摄像头通过 MediaPipe 识别手指位置 -> 命中 "dog"

Emma: "YES! Dog! D-O-G. Woof woof! Good job!"
```

记录: 单词匹配 3 轮, 正确率 100%

### 场景 7：投影绘本共读（Pro 版）

```
孩子拿纸质绘本放到桌上
摄像头识别封面 -> "The Very Hungry Caterpillar"

投影在绘本旁边投射:
  - 当前页英文句子(大字体)
  - 正在读的单词高亮
  - 毛毛虫小动画

Emma 读一页 -> 投影高亮当前单词
孩子可以在投影文字上指出不认识的词
摄像头识别指向 ->
Emma: "That word is strawberry. Can you say straw-berry?"

物理绘本(触觉) + 投影增强(视觉) + Emma 语音(听觉) = 三感官阅读
```

### 场景 8：画画 + 英语标注（Pro 版）

```
孩子在桌上画了一只猫
摄像头识别 -> "cat drawing"

投影在画旁边投射 "cat" 单词 + 动画小猫走向孩子画的猫

Emma: "Wow! A cat! C-A-T. Cat! Can you draw a ball for the cat?"

孩子继续画 -> 投影动画小猫追球
孩子得到"画被 AI 看见并回应了"的体验
```

### 场景 9：手势游戏 "影子猜词"（Pro 版）

```
Emma: "Can you make a bird with your hands?"

孩子用手做小鸟手影
摄像头识别 -> 投影投射小鸟剪影 + "bird" 单词

Emma: "A bird! Tweet tweet! B-I-R-D!"

孩子换手势(兔子) -> 投影兔子剪影
Emma: "A rabbit! Hop hop!"

集满 5 星 -> "You're a shadow master!"
```

## 5. 孩子端体验

### 5.1 交互通道（三级方案）

| 通道 | Basic | Mid | Pro |
|------|-------|-----|-----|
| **语音** | Yes | Yes | Yes |
| **LED 光环** | Yes | Yes | Yes |
| **物理按钮/旋钮** | Yes | Yes | Yes |
| **Emma 面部显示** | No | Yes (AMOLED) | Yes (投影) |
| **桌面互动投影** | No | No | Yes |
| **手势追踪** | 挥手唤醒 | 挥手唤醒 | 指向/拖拽/手影 |

### 5.2 交互设计（Basic 版）

孩子与 Camera Tutor 的交互通道：

没有屏幕，没有触控，没有任何需要在视觉界面中导航的内容。

### 5.2 LED 光环语言

| 颜色 | 动画 | 含义 |
|------|------|------|
| 蓝色呼吸 | 缓慢的明暗脉冲 | 📹 记录仪模式 — 我在观察，不说话 |
| 绿色常亮 | 恒亮 | 🧑‍🏫 导师模式 — 我在说英语，想加入吗？ |
| 绿色闪烁 | 快闪 2 次后常亮 | 🧑‍🏫 我有话要说（即将主动发起对话） |
| 黄色跳动 | 跳跃式明暗 | 🎮 游戏模式 — 来玩吧！ |
| 橙色旋转 | 顺时针流转 | 🤔 思考中（等待 AI 响应）|
| 紫色微光 | 极暗的紫色 | 🌙 睡前模式 — 安静 |
| 红色恒亮 | 恒亮 | 🔇 静音/休眠（镜头盖已盖或家长已禁用） |

### 5.3 物理交互

| 交互 | 操作 | 反馈 |
|------|------|------|
| 盖上镜头盖 | 手动推动镜头盖 | LED 变红色 → "See you later!" → 休眠 |
| 打开镜头盖 | 手动推开镜头盖 | LED 蓝色呼吸 → "I'm back!" → 恢复观察 |
| 旋转音量旋钮 | 手动旋转 | 播放一声轻微提示音，音量递增/递减 |
| 拍头顶/挥手 | 在摄像头前挥手 2 秒 | 进入 Show & Tell 模式："What do you want to show me?" |
| 大声说 "Emma!" | 语音唤醒 | "Yes? I'm here!" → 进入导师模式 |

### 5.4 声音角色：Emma 幼儿版

```
Emma (成人版)         →    Emma (幼儿版)
"Weather is nice."   →    "It's sunny! The sun is yellow and warm!"
"Please repeat."     →    "Let's say it together! Ready? One, two, three..."
"Tell me about..."   →    "What do you see? Is it a dog? Or a cat?"
"Good pronunciation."→    "Wow, you said that so well! High five!"
```

**语音特征**：
- 语速：比正常慢 20-30%
- 音调：稍高、更夸张的音调变化
- 词汇：最常用 500 词（对应 A1 级别）
- 句子长度：5-8 个词，最多 12 个词
- 重复：新词重复 3 次（不同语境）
- CS (Child-directed Speech) 特征：拖长元音、音调跨越大、关键词加重

## 7. 商业模式

### 7.1 收入模型

| 层级 | 硬件 | 服务 | 月费用 | 目标用户 |
|------|------|------|--------|---------|
| **基础版** | Camera Tutor 设备 | 本地 7B 模型（基础导师功能） | ¥0/月 | 所有用户 |
| **Pro 版** | Camera Tutor 设备 | 云端 30B 模型（更强教学、绘本精读、家长高级报告） | ¥29-39/月 | 重视效果的家长 |
| **Max 版** | Camera Tutor 设备 | Pro + AI 家长一对一辅导 + 儿童发音深度分析 | ¥69-99/月 | 高净值家庭 |

### 7.2 定价推导

| 维度 | 数据 |
|------|------|
| **硬件成本目标** | ¥500-800（量产 BOM）|
| **硬件售价** | ¥1,299-1,999（一次性） |
| **客户获取成本 (CAC)** | ¥200-400（母婴 KOL + 小红书种草） |
| **月活用户平均收入 (ARPMAU)** | ¥15-25 |
| **用户生命周期 (LTV)** | 24-36 个月（孩子 3-10 岁） |
| **LTV/CAC** | 3-5x（健康） |
| **毛利率（硬件）** | 40-50% |
| **毛利率（订阅）** | 70-80%（云端 API 成本约 ¥10-15/月/Pro 用户） |

### 7.3 分销渠道

| 阶段 | 渠道 | 策略 |
|------|------|------|
| **Alpha** | 直接招募（朋友/社群） | 免费赠送 5 台，换反馈 |
| **Beta** | 小红书/抖音种草 + 限时内测 | 限量 100 台，¥999 内测价，收集 UGC |
| **Launch** | 小米有品/京东众筹 + 母婴 KOL | 众筹目标 ¥20 万验证需求 |
| **Scale** | 京东/天猫旗舰店 + 母婴连锁（孩子王/爱婴室）+ 国际学校/培训机构 B2B | 正常销售 |
